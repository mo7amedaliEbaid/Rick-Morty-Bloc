# RickandMorty-GameofThrones-Bloc.
A Flutter app for( rick and morty) and (game of thrones) series,  
you can browse and search characters,   
based on free apis ,   
I used cubit ,blocprovider,flutter_offline packages
# Used Packages.
  dio: ^4.0.0    
  flutter_bloc: ^7.0.1    
  flutter_offline: ^2.0.0
